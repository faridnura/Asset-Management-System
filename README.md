**Cara Instalasi**
1. Ekstrak file project.
2. Jalankan composer install.
3. Jalankan npm install.
4. Salin .env.example menjadi .env.
5. Atur konfigurasi database pada .env.
6. Jalankan php artisan key:generate.
7. Jalankan php artisan migrate --seed (atau import database jika disediakan).
8. Jalankan php artisan storage:link.
9. Jalankan npm run dev.
10. Jalankan php artisan serve.
